<h1>🌟 Overview</h1>

This project is the **official business website developed for ABC Contracting & Construction**.

The website is designed to present the company’s **construction services, projects, and business information** in a professional and accessible way.

Visitors can explore the company's services and contact the company directly using the **integrated EmailJS contact system**.

The website is built using **React with Vite**, making it **fast, modern, and optimized for performance**.

<h1>📩 EmailJS Integration</h1>

The website uses **EmailJS** to allow customers to send inquiries directly from the contact form.

Workflow:

Customer fills contact form 📝  
⬇  
EmailJS processes the request 📡  
⬇  
Message is delivered to company email inbox 📩  

Benefits:

⚡ No backend server required  
📨 Instant message delivery  
🔒 Secure API-based email sending
